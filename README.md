# Proyek UAS Kecerdasan Buatan - Klasifikasi Kualitas Air

Proyek ini bertujuan untuk membangun model klasifikasi untuk menentukan apakah sampel air layak dikonsumsi atau tidak berdasarkan parameter kimia air.

## Anggota Kelompok
*   Azhari Ahmad Fauzani (2406107)
*   Luthfi Muhammad Fikri (2406003)

## Dataset
Dataset yang digunakan adalah *Water Potability Dataset* yang berisi 9 parameter kimia air.

## Hasil Evaluasi
*   **Decision Tree:** 63.26% Akurasi
*   **KNN:** 62.80% Akurasi

## Cara Menjalankan
1. Buka file `UAS_Uji_Kelayakan_Air_Minum.ipynb` di Google Colab.
2. Upload dataset `water_potability.csv`.
3. Jalankan semua *cell* secara berurutan.
